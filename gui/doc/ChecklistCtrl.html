<private, static> recalculateCategories(evt, ui)

Called when dropping (after dragging) a checklist items or checklist categories. This method is responsible for getting the order of items within each categry formatted in a way that the API will beable to save the new order. This medthod also initiates the request to the API to save the calculated order. Based on https://github.com/angular-ui/ui-sortable (ui-sortable) and http://jqueryui.com/draggable/ (jQuery-ui-draggable).
